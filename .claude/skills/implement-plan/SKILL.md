Implement the plan located at: $ARGUMENTS

## Implementation Process

1. **Read the entire plan first** - Understand the full scope before starting
2. **Follow the steps in order** - Execute each step as described
3. **Verify as you go** - Check that each step works before moving on
4. **Track progress** - Note what's been completed

## Guidelines

- Follow the plan's approach unless you discover a clear problem
- If a step can't be implemented as written, note why and adapt
- Keep changes focused on what the plan specifies
- Run tests after implementation if a testing strategy is defined
- Don't add features or improvements not in the plan

## On Completion

After implementing all steps:

1. List what was implemented
2. Note any deviations from the plan and why
3. Report the result of any tests run
4. Identify any follow-up work needed
